Librum ipsum adeat qui de ea methodo plura scire velit: ad institutum nostrum sufficit observare quanta intersit asfinitas inter Methodum hanc & Methodum Fluxionum seu differentialem Nam ut in Methodo differentiali, ad inveniendum differentiale ipsius x dignitatis xm, unum latus x convertendum est in differentiam dx; & ortum ducendum est in dignitatis Indicem m, ut sit m d x x m 1 differentiale quaesitum; sic in Methodo Incrementorum Ad inveniendum Incrementum facti hujusmodi x x x + n x x + 2 n, (ubi factores x, x + n, x + 2 n, (636) x + 2 n, sunt in progressione Arithmetica, cujus differentia communis est ipsius x Incrmentum datum n,) Factorum minimus-x convertendus est in Incrementum, & ortum ducendum est in numerum Factorum, ut sit 3 n x x + n x x + 2 n Incrementum quaesitum, numero Factorum in casu exposito existente 3. Sic etiam ipsius x x x + n Incrementum fit 2 n x x + n. 2. Incrementa etiam Reciprocorum hujusmodi Factorum inveniuntur per eandem regulam; hoc nempe observato, quod cum sit Divisio contrarium Multiplicationis, vice ablationis minimi Factorum, sit jam addendus alius factor adhuc uno Incremento major; item quod Factorum numerus sit scribendus cum signo negativo. Hoc pacto ipsius 1/x Incrementum fit 1 x n/x x x + n; ipsius 1/x x x + n Incrementum fit 2 x n/x x x + n x x + 2 n; & sic de aliis hujusmodi. Hoc facile probatur sumendo differentias inter Integralium valores duos continuos. 3. Insistendo vestigiis Methodi directae, hinc colliguntur praecepta Methodi inversae, quibus inveniuntur Integralia Incrementorum oblatorum. Applicetur enim Incrementum oblatum ad lateris Incrementum datum; addatur Factor adhuc uno Incremento minor, & applicetur ortum ad numerum Factorum sic auctorum. Sic e. g. oblato Incremento n x x x x + n x x + 2 n. sit primo x x x + n x x + 2 n; deinde x n x x x x + n x x + 2 n, addito Factore x n; denique x n x x x x + n x x + 2 n/4, quod est Integrale quaesitum. Hoc quidem ubi Factores sunt Multiplicantes; Ubi vero Factores occupant locum divisoris, mutatis mutandis, regula haec est, Applicetur Incrementum oblatum ad lateris incrementum ~ datum; rejiciatur Factorum (637) Factorum maximus, & applicetur ortum ad numerum Factorum relictorum cum signo negativo. Exempli gratia oblato Incremento n/x x x + n x x + 2 n' fit primo 1/x x x + n x x + 2 n, deinde 1/x x x + n, denique 1/-2 x x x x + n, seu -1/2 x x x + n, quod est Integrale quaesitum. 4.
